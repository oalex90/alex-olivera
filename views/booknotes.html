Book Notes

Instructions:

  • Enter a book title to add a new book.
  • Once a book title is added, you can add comments by clicking the book title and entering a value in the "New Comment" box and clicking the "Add Comment" button.
  • You can delete a book by clicking the "Delete Book" button, and you can delete all book titles by clicking the "Delete all books..." button.

Select a book to see it's details and comments

    API

    /booknotes/api

    GET

    Input: /booknotes/api
    Output: [{ "_id": "5d9cb3b8bf347911bd771a54", "title": "Matilda", "commentcount": 6 }, { "_id": "5d9cb47bbf347911bd771a55", "title": "Lord of the Rings", "commentcount": 0 } ]

    POST

    Input: "title": "Lord of the Rings"
    Output: { "_id": "5d9cb47bbf347911bd771a55", "title": "Lord of the Rings", }

    DELETE

    Input: None
    Output: { "success": "complete delete successful"}

    /booknotes/api/:id

    GET

    Input: /booknotes/api/5d9cb3b8bf347911bd771a54
    Output: { "_id": "5d9cb3b8bf347911bd771a54", "title": "Matilda", "comments": [ "comment one", "comment two", "comment three" ] }

    POST

    Input: "comment": "The middle section was incredibly structured."
    Output: { "_id": "5d9cb3b8bf347911bd771a54", "title": "Matilda", "comments": [ "comment one", "comment two", "comment three", "The middle section was incredibly structured." ] }

    DELETE

    Input: None
    Output: { "success": "delete successful"}
