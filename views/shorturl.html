URL Shortener Microservice

Instructions:

  • Enter a valid URL into the box below to receive a shortened URL that can be used to access the original URL.
  • Note: make sure to include http:// or https:// in your URL.

API

/shorturl

POST

Input: "url": "https://www.freecodecamp.org"
Output: { "original_url": "https://www.freecodecamp.org", "short_url": "1", }
