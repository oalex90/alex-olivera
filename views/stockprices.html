Nasdaq Stock Price Checker

Get single price and total likes

  • Enter a valid Nasdaq stock symbol to look up the current price and the current number of likes for that stock.
  • Check the "Like?" box to add a like to that stock. (Only 1 like per IP address is allowed)
Like?

Compare and get relative likes

  • Enter a valid Nasdaq stock symbol into each of the boxes to look up the current price and compare the number of likes for those stocks.
  • Check the "Like both?" box to add a like to each stock. (Only 1 like per IP address is allowed)
Like both?

API

Example Usage:

/stockprices/api?stock=goog
/stockprices/api?stock=goog&like=true
/stockprices/api?stock=goog&stock=msft
/stockprices/api?stock=goog&stock=msft&like=true

Example return:

{"stockData":{"stock":"GOOG","price":"786.90","likes":1}}
{"stockData":[{"stock":"MSFT","price":"62.30","rel_likes":-1},{"stock":"GOOG","price":"786.90","rel_likes":1}]}
